# 목적
brad frost의 `atomic-design` 방식으로 CDD(Component Driven Development)를 연습하고 storybook으로 문서화 하기 위해서.

# local에서 실행방법

## 스토리북 실행
```bash
npm run storybook
```

## vite로 App.tsx실행
```bash
npm run dev
```
